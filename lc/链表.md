#### 概念

##### 链表遍历方法：双指针法，对于新的链表，需要用一个虚拟节点，记得释放

#### 举例

##### 总结1

对于快慢指针获取中间指针，一般用如下函数

```c++
ListNode* get_middle(ListNode* head) {
	auto fast = head;
  auto slow = head;
  while (fast->next != nullptr &&fast->next->next != nullptr) {
    fast = fast->next->next;
    slow = slow->next;
  }
  return slow;

}
```

##### 总结2

* lru需要用头尾的虚拟节点来支持双链表
* 删除链表里所有重复的元素，需要哑结点和一个变量来保证
* 舍得用变量，千万别想着节省变量，否则容易被逻辑绕晕
* head 有可能需要改动时，先增加一个 假head，返回的时候直接取 假head.next，这样就不需要为修改 head 增加一大堆逻辑了。
